# FlowDesk - AI 智能客服 Landing Page

一个完整的 SaaS 产品 Landing Page，使用 React + Tailwind CSS 构建。

## 🎯 项目定位

面向 AI 智能客服产品的商业落地页。包含从导航、主视觉、功能展示、定价方案到 CTA 的完整营销页面结构。

## 🛠 技术栈

- **React 18** — 函数组件 + Hooks
- **Vite** — 构建工具
- **Tailwind CSS v4** — 原子化样式
- **CSS Animations** — 入场动画、悬停动效、渐变文字

## 📦 快速开始

```bash
npm install
npm run dev      # 启动开发服务器 http://localhost:5173
npm run build    # 构建生产版本
npm run preview  # 预览生产构建
```

## ✨ 核心功能

| 模块 | 功能 |
|------|------|
| 导航栏 | 固定顶部、毛玻璃背景、响应式菜单 |
| 主视觉 | 渐变文字标题、CTA 按钮、模拟聊天动画窗口 |
| 功能介绍 | 六宫格卡片布局、悬停上移动效 |
| 数据展示 | 4 栏统计数据（企业数、可用率、解决率、响应时间）|
| 定价方案 | 三档定价表、中间推荐高亮、功能对比 |
| CTA 区域 | 深色背景、二次转化入口 |
| 页脚 | 链接导航、版权信息 |

## 📱 响应式适配

- 手机（< 640px）：单列布局
- 平板（640px - 1024px）：双列布局
- 桌面（> 1024px）：完整多列布局

## 🎨 动画效果

- 页面加载淡入 + 上移动画（fadeInUp）
- 聊天气泡逐条延迟播放
- 功能卡片 hover 上移 + 阴影增强
- 按钮 hover 颜色过渡
- 定价卡片推荐版放大高亮

```
src/
```
