## 使用方法

1. 双击任意 .html 文件，浏览器打开预览
2. 用记事本/VS Code 打开文件，替换文字和图片即可
3. 所有模板都是响应式设计，手机/平板/电脑自动适配
4. 无需安装任何软件，纯 HTML + Tailwind CSS（CDN 加载）

## 技术说明

- HTML5 语义标签
- Tailwind CSS（CDN 在线加载，无需下载）
- CSS 入场动画（fadeInUp）
- 移动端适配（响应式断点）
- 干净注释，方便修改
